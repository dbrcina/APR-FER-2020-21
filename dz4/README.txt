Requirements: Java v15, Maven v3.6.3

1) Pozicionirati se u root direktorij - direktorij u kojem se nalazi src folder i pom.xml datoteka
2) Izvrsiti naredbu mvn compile
3) Izvrsiti naredbu java -cp target/classes hr.fer.zemris.apr.hw04.Zadatak1 param
		param: f1-decimal, f3-decimal, f6-decimal, f7-decimal, f1-binary, f3-binary, f6-binary, f7-binary
	
	Izvrsiti naredbu java -cp target/classes hr.fer.zemris.apr.hw04.Zadatak2 param
		param: f6, f7
		
	Ostali demo programi sluze za generiranje podataka koji se onda obraduju pomocu pythona i rezultati se mogu pronaci u apr_dz4.ipynb.
